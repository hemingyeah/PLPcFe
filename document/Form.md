## 工单字段类型：
- customer          客户
- taskNo            工单编号
- planTime          计划时间

- relationCustomer  客户关联
- relationProduct   产品关联

- cascader          多级菜单
- attachments       自定义附件
- address           地址

- level             优先级        alias select
- serviceType       服务类型      alias select
- serviceContent    服务内容      alias select

- description       描述         alias textarea
